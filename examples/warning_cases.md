# Warning Cases — DDC Examples

These cases illustrate how ignoring user dignity leads to
real consequences. Numbers and legal details are omitted
where verification is incomplete. Marked with [verify] where
independent confirmation is recommended before citing.

---

## English

### Epic Games (Fortnite)
**Pattern:** Manipulative purchase flows targeting minors.
Disguised costs, accidental purchases made easy to trigger.
**Consequence:** FTC settlement including COPPA violations
and dark pattern charges. [verify: exact amounts and breakdown]
**Structure:** Exploit → Regulatory action → Forced settlement

### Amazon Prime
**Pattern:** Cancellation obstruction. Multi-step, confusing
unsubscribe flows designed to exhaust user intent.
**Consequence:** FTC lawsuit. [verify: current status and outcome]

### Cambridge Analytica
**Pattern:** Consent obfuscation. Users did not meaningfully
agree to the scope of data collection and political targeting.
**Consequence:** Collapse of the organization. [verify: full timeline]
**Structure:** Data exploitation → Public exposure → Trust collapse → Dissolution

### National Public Data / 23andMe
**Pattern:** Inadequate data protection relative to the
sensitivity of data collected. [verify: specific dark pattern

### The Pattern Behind the Pattern

These cases share a structure:

1. Short-term gain through user exploitation
2. Accumulation of hidden risk (legal, reputational, human)
3. Triggering event (lawsuit, breach, exposure)
4. Collapse disproportionate to the original gain

Dignity ignored is not profit deferred. It is risk compounded.

複雑で多段階の退会フロー。
**結果:** FTC提訴。[要検証：現在の訴訟状況と結末]

政治的ターゲティングの実態に、実質的に同意していなかった。
**結果:** 組織の崩壊。[要検証：全経緯]

保護が不十分だった。[要検証：ダークパターンの直接的関与、

**構造:** データ漏洩 → 法的責任 → 企業価値の崩壊

### パターンの背後にあるパターン

これらの事例には共通の構造がある：

1. ユーザーの搾取による短期的な利益
2. 隠れたリスクの蓄積（法的・評判的・人間的）
3. 引き金となる出来事（訴訟、漏洩、暴露）
4. 元の利益に不釣り合いな崩壊

尊厳を無視することは、利益を先送りにすることではない。
リスクを複利で積み上げることだ。
